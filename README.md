# nopolicybot — умный политмонитор для Telegram

**nopolicybot** — это телеграм‑бот, который:

- следит за политическими и токсичными сообщениями в чате и аккуратно вмешивается, когда это нужно;
- ведёт себя как живой, язвительный собеседник (умеет злиться, уходить в игнор и мириться);
- строит **дерево связей** между участниками: кто с кем, на какие темы и в каком тоне общается;
- даёт администратору **читабельные дайджесты и визуальную аналитику**, а не сухие таблички.

Проект предназначен для закрытых чатов и сообществ, где важны и свобода общения, и контроль над политикой/конфликтами.

---

## Основные возможности

- **Модерация политики и токсичности**
  - анализ текста и изображений на политический/конфликтный контент;
  - гибкая настройка порогов и стиля реакций: от мягких лайков до жёстких замечаний;
  - защита от спама замечаниями (интервалы, счётчики, режим «через раз» и т.п.).

- **Живой характер бота**
  - разные режимы тона (forgive/angry/rage/balanced) в зависимости от истории общения;
  - умеет послать пользователя и **включить паузу общения** (игнор на заданное время);
  - снимает паузу, если в сообщении распознаёт извинение или примирение (даже неявное).

- **Дерево связей и социальная аналитика**
  - строит граф связей по reply‑to: кто кому отвечает и насколько активно;
  - визуализирует связи (аватарки + линии) и показывает список пар с метриками;
  - считает тон, темы, confidence, «алерты» (новая связь, рост активности, всплеск токсичности);
  - умеет подсветить риск конфликтов и роли участников (связующий, эксперт, медиатор, провокатор).

- **Дайджесты и «вопрос дня»**
  - генерирует **дайджест чата**: ключевые диалоги, темы, активность, тон;
  - может автоматически или вручную задавать пользователям персональные «вопросы дня» (через ансамбль моделей).

- **Память и объяснимость**
  - хранит портреты пользователей и архив расшифровок картинок;
  - записывает «объяснения» своих решений, которые можно просмотреть в админке.

## Быстрый старт

## Что нужно

1. **Telegram-бот**  
   Создайте бота через [@BotFather](https://t.me/BotFather): команда `/newbot`, затем скопируйте выданный токен.

2. **API ключ ИИ**  
   - OpenAI: ключ с [platform.openai.com](https://platform.openai.com/api-keys)  
   - Или любой сервис с OpenAI-совместимым API (базовый URL задаётся в `.env`).

## Установка

```bash
cd telegram-political-monitor-bot
python -m venv venv
venv\Scripts\activate   # Windows
# или: source venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
```

## Настройка

1. Скопируйте пример конфига и отредактируйте:
   ```bash
   copy config.example.env .env
   ```
2. В `.env` укажите:
   - `TELEGRAM_BOT_TOKEN` — токен от BotFather  
   - `OPENAI_API_KEY` — ключ API ИИ  
   - **База:** PostgreSQL рекомендуется. Пример с Docker: в корне `docker compose up -d postgres` (см. `docker-compose.yml`), в `.env` те же `POSTGRES_USER` / `POSTGRES_PASSWORD` / `POSTGRES_DB`, что у контейнера, и **`POSTGRES_HOST=127.0.0.1`** если бот на хосте. Явный DSN: `docs/DATABASE_POSTGRES.md`.
   - При необходимости: `OPENAI_BASE_URL`, `OPENAI_MODEL`
   - Для мульти-модельных ответов бота (по обращениям к боту):  
     `OPENAI_REPLY_MODELS=model1,model2,model3`  
     Пример для OpenRouter:  
     `OPENAI_REPLY_MODELS=stepfun/step-3.5-flash:free,google/gemini-2.0-flash-exp:free,openai/gpt-4o-mini`

## Запуск

```bash
python bot.py
```

## Админ-панель

Веб-интерфейс для просмотра статистики пользователей, портретов и рангов.

```bash
python admin_app.py
# или: run_admin.bat
```

Открыть в браузере: http://127.0.0.1:5000

В `.env` можно задать:
- `ADMIN_PASSWORD` — пароль для входа (если не задан — вход без пароля, только localhost)
- `ADMIN_SECRET_KEY` — секрет для сессий (обязательно в продакшене)
- `ADMIN_HOST`, `ADMIN_PORT` — хост и порт (по умолчанию 127.0.0.1:5000)

**Портрет из архива:** на странице пользователя кнопка «Построить портрет» анализирует до 1000 сообщений, которые бот прочитал в чате, и строит подробный портрет (психологический, профессиональный, политический) через ИИ.

## Использование в группе

1. Добавьте бота в группу (как участника).
2. Выдайте боту право **читать сообщения** (в настройках группы или правах администратора).
3. В группе напишите боту `/start` — он ответит приветствием.
4. Бот будет анализировать последние сообщения в чате и при обнаружении политической темы отправлять вежливое замечание в ответ на сообщение.

Логика анализа и формулировки замечаний задаётся в `ai_analyzer.py` (системный промпт). При желании можно изменить тон или критерии «политичности».
