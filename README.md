# Telegram-бот: модерация политических тем через ИИ

Бот следит за перепиской в групповых чатах и вежливо делает замечания, когда участники переходят на политические темы. Анализ и формулировка замечаний выполняются через ИИ (OpenAI или совместимый API).

## Что нужно

1. **Telegram-бот**  
   Создайте бота через [@BotFather](https://t.me/BotFather): команда `/newbot`, затем скопируйте выданный токен.

2. **API ключ ИИ**  
   - OpenAI: ключ с [platform.openai.com](https://platform.openai.com/api-keys)  
   - Или любой сервис с OpenAI-совместимым API (базовый URL задаётся в `.env`).

## Установка

```bash
cd telegram-political-monitor-bot
python -m venv venv
venv\Scripts\activate   # Windows
# или: source venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
```

## Настройка

1. Скопируйте пример конфига и отредактируйте:
   ```bash
   copy config.example.env .env
   ```
2. В `.env` укажите:
   - `TELEGRAM_BOT_TOKEN` — токен от BotFather  
   - `OPENAI_API_KEY` — ключ API ИИ  
   - При необходимости: `OPENAI_BASE_URL`, `OPENAI_MODEL`
   - Для мульти-модельных ответов бота (по обращениям к боту):  
     `OPENAI_REPLY_MODELS=model1,model2,model3`  
     Пример для OpenRouter:  
     `OPENAI_REPLY_MODELS=stepfun/step-3.5-flash:free,google/gemini-2.0-flash-exp:free,openai/gpt-4o-mini`

## Запуск

```bash
python bot.py
```

## Админ-панель

Веб-интерфейс для просмотра статистики пользователей, портретов и рангов.

```bash
python admin_app.py
# или: run_admin.bat
```

Открыть в браузере: http://127.0.0.1:5000

В `.env` можно задать:
- `ADMIN_PASSWORD` — пароль для входа (если не задан — вход без пароля, только localhost)
- `ADMIN_SECRET_KEY` — секрет для сессий (обязательно в продакшене)
- `ADMIN_HOST`, `ADMIN_PORT` — хост и порт (по умолчанию 127.0.0.1:5000)

**Портрет из архива:** на странице пользователя кнопка «Построить портрет» анализирует до 1000 сообщений, которые бот прочитал в чате, и строит подробный портрет (психологический, профессиональный, политический) через ИИ.

## Использование в группе

1. Добавьте бота в группу (как участника).
2. Выдайте боту право **читать сообщения** (в настройках группы или правах администратора).
3. В группе напишите боту `/start` — он ответит приветствием.
4. Бот будет анализировать последние сообщения в чате и при обнаружении политической темы отправлять вежливое замечание в ответ на сообщение.

Логика анализа и формулировки замечаний задаётся в `ai_analyzer.py` (системный промпт). При желании можно изменить тон или критерии «политичности».
