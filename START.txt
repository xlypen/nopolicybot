  КАК ЗАПУСТИТЬ БОТА (на сервере или у себя)
═══════════════════════════════════════════════════════════

1. Скопируйте всю папку telegram-political-monitor-bot на сервер.

2. Откройте файл .env и вставьте свои данные (сохраните):
   • TELEGRAM_BOT_TOKEN = токен от @BotFather
   • OPENAI_API_KEY = ваш ключ DeepSeek (sk-a7e9...)

3. Запуск:

   На Linux/сервере:
     chmod +x run.sh
     ./run.sh

   На Windows:
     Дважды клик по run.bat
     или в командной строке: run.bat

   Или вручную:
     python -m venv venv
     venv\Scripts\activate   (Windows)  или  source venv/bin/activate  (Linux)
     pip install -r requirements.txt
     python bot.py

4. Бота добавьте в группу в Telegram и дайте право читать сообщения.

Готово. Бот будет анализировать переписку через DeepSeek и делать
замечания при политических темах.
